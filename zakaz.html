Информация о доставке

Самое экономичное решение для вас

Стандартная доставка, включая разгрузку

Доставка грузов осуществляется с 08:00 до 17:00. Полный груз может содержать до 13 паллет или биг-бэгов.

Доставка крупногабаритных грузов без разгрузки

Покупатель должен предоставить разгрузочное оборудование для доставки товара. Одна полная загрузка включает до 21 паллеты. Перед доставкой покупатель должен убедиться, что по указанному им адресу может проехать транспортное средство для крупногабаритных грузов.

Без доставки

Заказы можно получить в рабочие дни с 09:00 до 17:00 по адресу: Атбалс иела 5, Латгальский пригород, Рига, LV-1063. Чтобы договориться о получении, пожалуйста, позвоните по телефону +371 26377636.
